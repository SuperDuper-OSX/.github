## Overview

Backup software earns trust through reliability over time — the confidence that when a drive fails, the backup will work exactly as expected to restore a complete, bootable system state without surprises or additional recovery steps. <a href="#">SuperDuper</a> from Shirt Pocket has built this trust across more than two decades of continuous development and active use by macOS users who depend on bootable backups as the foundation of their data protection strategy. The application's focus on doing one thing — creating reliable, complete, bootable drive clones — with maximum simplicity and reliability has kept it the preferred backup tool for users who prioritize dependability over feature breadth.

The bootable clone produced by <a href="#">SuperDuper</a> is functionally indistinguishable from the original drive it was copied from — it starts the system, loads the operating system, and presents the complete application and data environment that existed at the time of the last backup, without requiring any restoration process. This immediacy of recovery is the key practical advantage of bootable cloning over snapshot-based backup systems: instead of waiting for a restoration process to complete before the system is usable, the backup drive can be used directly, allowing work to continue immediately while the primary drive situation is addressed. The <a href="#">Smart Update</a> capability makes incremental backups fast enough to run on a schedule — copying only files that have changed since the last backup rather than duplicating the entire drive each time — enabling the frequent backup cadence that meaningful protection requires.

The Sandbox feature in <a href="#">SuperDuper</a> provides a practical capability for cautious system management that goes beyond simple backup. A Sandbox is a bootable clone that shares user data with the original system but has its own independent copy of the system software, allowing operating system updates and major software installations to be tested in the Sandbox environment before being applied to the production system. If the update causes problems in the Sandbox, the production system remains unaffected and the Sandbox can simply be rebuilt. This <a href="#">risk-free testing environment</a> is particularly valuable before major macOS version upgrades or before installing software that modifies system components. <a href="#">SuperDuper OSX</a> is the reliable, simple bootable backup solution that has earned the trust of demanding macOS users for over two decades.

## Key Features

- Create fully <a href="#">bootable drive clones</a> that can start the system directly after primary drive failure
- Smart Update copying only <a href="#">changed and new files</a> for fast incremental backup operations
- Scheduled automatic <a href="#">backup tasks</a> running while you sleep for continuous protection
- Sandbox creation for <a href="#">testing system updates</a> and software on a clone before applying to the main drive
- Support for <a href="#">APFS and HFS+</a> source and destination volumes on all current hardware
- Copy scripts defining <a href="#">exactly what is included</a> and excluded from each backup operation
- Erase-then-copy and <a href="#">Smart Update modes</a> for different backup strategy requirements
- Simple, clear <a href="#">progress interface</a> with detailed log of all copying operations
- Disk <a href="#">repair and verification</a> steps during backup ensuring destination integrity
- Support for <a href="#">external drives, NAS</a>, and network volumes as backup destinations
